불안형 애착은 사랑이 부족해서 생기는 문제가 아니라, 관계의 불확실성을 크게 감지하는 마음의 습관과 더 가깝습니다.

답장이 늦어지는 짧은 순간에도 버려질지 모른다는 상상이 앞서면, 상대의 실제 행동보다 내 해석이 관계를 더 흔들 수 있습니다.

패턴을 이해하면 감정을 억누르지 않아도 됩니다. 대신 감정과 사실을 분리해 보는 연습이 가능해집니다.
